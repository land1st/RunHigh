# RunHigh 🏃
> *One run at a time*

모바일 최적화 러닝 훈련 기록 앱

## 기능
- 월간 훈련 계획 생성 (조깅 / 포인트 훈련 / LSD)
- 훈련 완료 기록 (거리 / 시간 / 페이스 / 심박수 / 케이던스 / 컨디션 / 메모 / 사진)
- 훈련 상세 화면 + B스타일 공유 이미지 합성
- 월간 통계 (계획 vs 실제 비교)

## 실행 방법

```bash
npm install
npm run dev
```

## 배포 (GitHub Pages)

```bash
npm run deploy
```

## 기술 스택
- React 18
- Vite
- Canvas API (이미지 합성)
